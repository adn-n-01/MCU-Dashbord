# Marvel Cinematic Universe (MCU) Dashboard – Power BI

## Project Overview

This project is an interactive Power BI dashboard that analyzes data from the Marvel Cinematic Universe (MCU) movies. The dashboard provides insights into movie performance, ratings, and other key metrics using visually engaging charts and indicators.

## Features

* Interactive Power BI dashboard
* Analysis of MCU movies
* Box office performance visualization
* Movie ratings and popularity insights
* Clean and easy-to-understand visualizations

## Tools Used

* Microsoft Power BI
* Data Visualization Techniques
* Movie Dataset (MCU related data)

## File Included

* `mcu_dashboard.pbix` – Power BI dashboard file

## Purpose

This project was created for learning and demonstrating data visualization and dashboard design skills using Power BI.
